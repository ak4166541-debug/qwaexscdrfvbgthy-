# Сайт который поможет нам с распределением финансов 
## Описание проекта
Мы создали не просто сайт для учета расходов, а комплексную образовательно-аналитическую платформу, которая помогает пользователям не только видеть, куда уходят деньги, но и понимать, как ими эффективно управлять, чтобы достигать своих финансовых целей. 
## Туториал 
1) Переходим по ссылке 
https://myfinancialcompass.website/
2) Впишите ваш месячный доход
3) После этого Вам будет показано как лучше распределить финансы на месяц
4) Ниже будет распологатся стратегии и риски
5) Ещё ниже будет визуализация вашего бюджета на год

## Технические требования 
Для работы проекта вам понадобится:
1) Любой девайс (телефон, планшет, компьютер...)
2) Любой браузер (Safari, Яндекс, Google...)
3) Сайт https://myfinancialcompass.website/
